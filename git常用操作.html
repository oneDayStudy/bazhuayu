常用命令

pwd:查看所在目录结构

git add . :添加最近所有到暂存区

git commit -m 'mess' :提交仓库

git status:查看暂存区和仓库状态

git diff file:查看file文件的最新变化

q :退出查看状态

git log:穿梭前，用git log可以查看提交历史，以便确定要回退到哪个版本。

git reflog: 要重返未来，用git reflog查看命令历史，以便确定要回到未来的哪个版本。

git reset （版本号-前几位即可）: 返回到指定的过去或未来的具体版本

git reset head file(如果add 但未commit):把file 从暂存区退回到工作区 然后再git checkout -- file :把工作区清理干净（既可完全把写错并且add-但没提交的文件恢复原样）

git checkout -- file:把工作区内容恢复（写错但未提交）2:切换到file分支

git checkout -b dev: 创建dev分支并切换到dev分支 相当于两条命令:$ git branch dev $ git checkout dev

git branch:查看当前分支 （会列出所有分支）*的是当前分支

git branch (name) :创建分支：git branch name

git branch -d name 删除分支

git merge dev：把dev分支合并到当前分支里

find ./ -type f -size +102400k:查找当前文件大于100m 的文件

github 限制在1G，单个文件不能超过100M，有50M的文件，就会警告了。 所以在上传代码时要记得把一些大的tag文件去掉，如SourceInsight工程文件 ctags等 可通过以下命令查找超过100M的文件 find ./ -type f -size +102400k
