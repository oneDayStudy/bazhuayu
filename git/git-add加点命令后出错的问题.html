1： 当我 执行 git add somefile 的时候，出现 如下 错误： If no other git process is currently running, this probably means a git process crashed in this repository earlier. Make sure no other git process is running and remove the file manually to continue 解决方法： rm -f ./.git/index.lock ====================== 2：window下运行git add. 时 提示warning: LF will be replaced by CRLF in .idea/compiler.xml. 这是Linux下换行符LF在window下为CRLF 解决方法为：运行如下代码git config core.autocrlf false即可
